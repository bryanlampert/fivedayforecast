# Five Day Forecast

#### By Bryan Lampert

---

#### Setup

* install dependencies
```npm install```

* run in dev mode
```npm run dev```

* run the app in the browser at
```localhost:8080```

#### Stack
```
* Vue.js
* Javascript
* HTML
* CSS
```
Scaffolding built using the Vue CLI.

##### Additional dependencies
```
* Axios
* Moment
```
